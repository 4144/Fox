Скрипты для помощи в поисках новых настроек:
* `fx-get-dist` - скачивает дистрибутивы Firefox и присваивает им однообразные имена, чтобы не возникало путаницы в билдах разных версий, ОС, разрядностей и локалей.
* `fx-unpack-prefs` - вытаскивает из дистрибутивов файлы с дефолтныим настройками.
* `fx-diff-prefs` - сравнивает дефолтные настройки разных версий.
